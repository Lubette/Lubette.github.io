使用工具快速配置VSCode（Visual Studio Code）的C/C++开发环境！

在之前的一篇推荐软件的文章里面，我推荐大家去使用VSCode这个轻量级的软件去写程序

新手小白碰到VSCode的配置肯定觉得麻烦或者不会，现在有了一个软件可以快速配置我们VSCode的C/C++开发环境

好了我们现在进入正题吧

操作流程

先下载配置工具，这需要我们进入Github网站，如果无法加载的话可以使用下面的网盘

Github下载地址：https://github.com/SDchao/AutoVsCEnv_WPF/releases/latest

国内网盘地址：https://sdchao.lanzoum.com/iFGmo0s82ybe

网盘地址一般来说会相对快一点

有什么问题的话可以看顶层文件夹里面的 README_inProject.md 文件
