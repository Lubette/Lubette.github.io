我的世界基岩服务器开服教程

我的世界基岩版的开服流程与Java版的开服教程有些许区别，所以我在这里简略的写一下关于基岩版开服的过程

安装JDK

Java Downloads|Oracle 中国 选择一个适合自己的版本下载

下载服务端

NukkitX服务端
这个是在我视野里出现最多次的基岩插件端服务器，点击链接下载它

启动服务端

执行下面这个指令

java -jar NukkitX-general-build1095.jar

你会看到这个画面，选择你想使用的语言，我选择中文（简体）所以我输入chs回车

出现下面这个界面就代表你的服务端已经正式启动了，连上对应的端口就能开始游戏了

本教程到此结束，别问我为啥这么简短，因为主打的就是一个短小精悍
